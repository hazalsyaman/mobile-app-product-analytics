# 📱 Mobile App Product Analytics

A portfolio project that simulates user behavior in a fictional mobile application and analyzes key product metrics using Python.

## Project Overview

This project recreates a realistic mobile app analytics workflow by generating synthetic user activity and calculating common product metrics used by Product Analysts and Data Analysts.

The fictional mobile application, **FocusFlow**, is designed to simulate user engagement, subscription behavior, and retention over time.
